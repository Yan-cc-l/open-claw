## 学习建议

1. **按顺序学习** - 课程内容由浅入深，建议按lecture顺序学习
2. **动手实践** - 每节课都有代码示例，建议自己敲一遍
3. **完成作业** - 课程有配套作业，可以检验学习效果
4. **看视频** - MIT提供了完整的视频讲座
